├─────────────────┼───────────────────┼──────────────────────────────────────────────────────────────┤

│ post-commit     │ Após cada commit  │ Invalida cache do ProjectStatusLoader (async, não bloqueia)  │

Repositórios com hooks configurados:
  ~/.husky/ (global)                          → pre-commit + pre-push
  ~/aios-core/                                → pre-commit + pre-push + post-commit
  ~/Documents/aios-control-center/            → pre-commit + pre-push + post-commit
  ~/Documents/ecosistema-ia-avancada-pt/...  → setup via setup-git-hooks.sh

Para novo repositório:
