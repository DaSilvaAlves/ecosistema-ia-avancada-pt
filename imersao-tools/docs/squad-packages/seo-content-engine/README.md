# SEO & Content Engine

Auditoria SEO completa com action items prioritizados: análise técnica, keywords, on-page, content gaps.

## Requisitos

- Claude Code instalado (`npm install -g @anthropic-ai/claude-code`)
- Chave API Anthropic activa

## Instalação

1. Extrai o ZIP para a pasta raiz do teu projecto
2. Abre o terminal e executa `claude`
3. Invoca a skill: `/market-seo`

## O que faz

- Auditoria técnica SEO completa
- Análise de keywords e oportunidades
- Optimização on-page com recomendações
- Identificação de content gaps
- Action items prioritizados por impacto

## Exemplo prático

```
$ claude
> /market-seo
> Faz uma auditoria SEO completa do site example.com
```
