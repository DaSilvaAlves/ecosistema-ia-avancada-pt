# Clone Alex Hormozi — Consultor Estrategico

Clone da mente do Alex Hormozi — autor de "$100M Offers" — adaptado ao mercado portugues. Nao e uma traducao. E o Hormozi a pensar com precos PT, fiscalidade PT, e a realidade de um mercado de 10 milhoes de pessoas.

Inclui:
- 17 frameworks operacionais (Mosy 6, VSSL, Grand Slam Offers, Value Equation, 5x Upsell, AD FOCUS, etc.)
- 27 heuristicas de decisao
- 9 padroes decisorios
- 7 historias experienciais
- Camada de adaptacao Portugal (precos, fiscalidade, cultura, canais)

## Requisitos

- [Claude Code](https://claude.ai/download) instalado
- Plano Claude (Max ou Pro)
- Terminal (VS Code, Windows Terminal, iTerm)

## Instalacao

1. Extrai o ZIP para a pasta raiz do teu projecto
2. Os ficheiros ficam em `.claude/agents/` automaticamente
3. Abre o terminal na pasta do projecto
4. Executa `claude` para iniciar o Claude Code
5. Activa o clone com `@alex-hormozi`

## Exemplo pratico

```
$ claude
> @alex-hormozi
> Tenho um negocio de servicos que fatura 3.000 EUR/mes. Como escalo para 10.000?
```

## Comandos disponiveis

| Comando | O que faz |
|---------|-----------|
| `*diagnostico` | Mosy 6 completo — diagnostica o constraint do teu negocio |
| `*oferta` | Grand Slam Offer — cria uma oferta irrecusavel |
| `*pricing` | Estrategia de pricing com 5x Upsell e adaptacao PT |
| `*vssl` | Cria guiao VSSL (Video Sales System Letter) |
| `*avatar` | Analise profunda do teu avatar ideal |
| `*escala` | Plano de escala: constraint → leverage → compounding |
| `*help` | Todos os comandos |
| `*exit` | Sair do modo Hormozi |
