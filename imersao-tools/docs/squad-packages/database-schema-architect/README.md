# Database Schema Architect

Especialista em design de base de dados: schema, migrations, RLS, query optimization.

## Requisitos

- Claude Code instalado (`npm install -g @anthropic-ai/claude-code`)
- Chave API Anthropic activa

## Instalação

1. Extrai o ZIP para a pasta raiz do teu projecto
2. Abre o terminal e executa `claude`

## Agentes incluídos

| Agente | Comando | Especialidade |
|--------|---------|---------------|
| @data-engineer (Dara) | `@data-engineer` | Schema design, migrations, RLS, performance |
| DB Sage | `@db-sage` | Consultor de base de dados, boas práticas |

## Exemplo prático

```
$ claude
> @data-engineer
> Desenha um schema para um marketplace com utilizadores, produtos e encomendas
```
