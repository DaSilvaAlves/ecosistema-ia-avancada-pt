# FASE 4 — Ponto Socorro noturno (Sabado 22:00-23:30)

> Sessao remota para desbloquear quem ficou para tras.
> Responsavel: Eurico

---

## Objectivo

Garantir que NINGUEM chega ao domingo sem projecto funcional. Quem ficou preso no sabado tarde tem aqui a oportunidade de resolver.

---

## Formato

| Campo | Valor |
|-------|-------|
| Canal | Google Meet |
| Duracao | 90 min (22:00-23:30) |
| Tipo | Suporte individual rotativo |
| Quem participa | So quem precisa (voluntario) |

---

## Roteiro

| Momento | Duracao | Accao |
|---------|---------|-------|
| Check-in | 10 min | "Quem esta? Qual e o bloqueio?" |
| Suporte 1:1 | 60 min | Eurico resolve com cada aluno (screen share) |
| Wrap-up | 20 min | "Para amanha: foco em [X]" |

---

## Ferramentas

| Ferramenta | Papel |
|------------|-------|
| Google Meet | Videochamada com screen share |
| Claude Code | Eurico corre no seu terminal para ajudar alunos |
| WhatsApp | Coordenacao rapida ("entra no Meet") |

---

## Tipos de bloqueio comuns

| Bloqueio | Solucao |
|----------|---------|
| "Nao consegui fazer deploy" | Eurico guia: git push + vercel --prod |
| "O codigo tem bugs" | Claude Code: "Corrige este erro: [mensagem]" |
| "Nao sei o que mais adicionar" | Eurico sugere 2-3 features concretas |
| "A ideia mudou" | Pivotar — Claude Code regenera com novo briefing |
| "Nao tenho ideia nenhuma" | Eurico ajuda a articular — usar prompts guiados |

---

## Output desta fase

| Output | Formato |
|--------|---------|
| Todos os alunos com projecto funcional | Deploy feito ou quase |
| Lista de tarefas para domingo | WhatsApp ou nota pessoal |
