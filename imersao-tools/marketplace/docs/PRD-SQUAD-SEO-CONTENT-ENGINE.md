| Skill | Comando | O que faz |
|-------|---------|-----------|
| SEO Audit | `/market-seo` | Auditoria tecnica SEO completa, keywords, on-page, content gaps, action items prioritizados |

## 3. Para quem e

Quem quer melhorar o SEO do seu site — analise tecnica, oportunidades de keywords, optimizacao on-page e identificacao de lacunas de conteudo.

---

## 4. Requisitos

- Claude Code instalado (`npm install -g @anthropic-ai/claude-code`)
- Chave API Anthropic activa (console.anthropic.com)
- Terminal (VS Code, Windows Terminal, iTerm)

---

## 5. Guia de instalacao

### Passo 1 — Descarregar

Clica em "Instalar" na pagina do produto no marketplace.

### Passo 2 — Extrair

Extrai o ZIP para a pasta raiz do teu projecto. Os ficheiros ficam em `.claude/skills/` automaticamente.

### Passo 3 — Abrir terminal

Abre o terminal na pasta do teu projecto.

### Passo 4 — Iniciar Claude Code

```
claude
```

### Passo 5 — Activar

```
/market-seo
```

---

## 6. Exemplo pratico

```
$ claude
> /market-seo
> Faz auditoria SEO completa do meu site: https://meurestaurante.pt
```
