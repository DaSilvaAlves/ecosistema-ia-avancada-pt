# Data Intelligence Squad

3 especialistas em dados: orquestração, engenharia de dados e análise.

## Requisitos

- Claude Code instalado (`npm install -g @anthropic-ai/claude-code`)
- Chave API Anthropic activa

## Instalação

1. Extrai o ZIP para a pasta raiz do teu projecto
2. Abre o terminal e executa `claude`

## Agentes incluídos

| Agente | Comando | Especialidade |
|--------|---------|---------------|
| Data Chief | `@data-chief` | Orquestração de data intelligence |
| @data-engineer (Dara) | `@data-engineer` | Schema, migrations, RLS, queries |
| @analyst (Alex) | `@analyst` | Pesquisa, análise de mercado, investigação |

## Exemplo prático

```
$ claude
> @data-chief
> Preciso de uma estratégia de dados para um marketplace B2B
```
