# WhatsApp Copy Rules — [IA]AVANÇADA PT

## Quando esta regra se aplica

Sempre que o utilizador pedir para escrever copy de resposta WhatsApp para leads da comunidade [IA]AVANÇADA PT.

## Referência obrigatória

Antes de escrever qualquer resposta WhatsApp:
1. Consultar os templates em `imersao-tools/docs/copy-templates/whatsapp-responses.md`
2. Identificar o perfil do lead (experiente, iniciante, desconfiado, profissional, brasileiro)
3. Escolher o template adequado e adaptar ao contexto

## Tom e estilo do Eurico (voz do fundador)

| Elemento | Regra |
|----------|-------|
| Tom | Directo, praticante — nunca vendedor ou professor |
| Estilo | Conversa entre pares, curto, sem floreados |
| Propósito | Sempre presente — "construímos ferramentas e damos condições para ter sucesso" |
| Comunidade | Posicionar como ajuda mútua — "todos se ajudam, quem sabe mais puxa" |
| Membro | É sempre o herói — a comunidade é o guia |
| CTA | Link directo, sem pressão — "entra e vê por ti" |

## Argumentos-chave (ordem de prioridade)

1. Propósito: ferramentas reais + condições para sucesso
2. Ajuda mútua: rede onde todos contribuem
3. Ferramentas concretas: pipeline de 7 ferramentas
4. Gratuito: sem truques, sem cartão
5. Formações práticas: 8 formações operacionais

## Proibições

| Proibido | Alternativa |
|----------|-------------|
| "curso" | "formação" |
| "fácil" | "prático", "operacional" |
| "automático" | "construir", "implementar" |
| "revolucionário" | "real", "concreto" |
| "garantido" | "comprovado", "testado" |
| Tom de vendas | Tom de conversa entre pares |
| Mensagens longas (>10 linhas) | Máximo 6-8 linhas |
| PT-BR na resposta | Sempre PT-PT, mesmo para leads BR |

## Workflow

1. Identificar perfil do lead pela mensagem + número de telefone
2. Consultar template adequado em `whatsapp-responses.md`
3. Adaptar ao contexto (nome, detalhe da mensagem)
4. Apresentar ao utilizador para aprovação
5. Se lead relevante, sugerir guardar em memória para follow-up
