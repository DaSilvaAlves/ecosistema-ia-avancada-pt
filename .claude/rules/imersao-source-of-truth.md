# Imersao — Fonte da Verdade

## REGRA INEGOCIAVEL

Antes de QUALQUER trabalho na imersao (PRD, kit, guia, roteiro, conteudo, horarios), o agente DEVE:

1. LER `imersao-tools/comunidade/todolist.html` — e a pagina LIVE, publicada, que os participantes veem
2. Extrair horarios, fases, ferramentas e estrutura DESTE ficheiro
3. So depois consultar docs internos (ORIGINAI, roteiro, kit) como material complementar

## Hierarquia de fontes

| Prioridade | Fonte | O que contem |
|------------|-------|-------------|
| 1 (maxima) | `comunidade/todolist.html` | Horarios, fases, ferramentas, estrutura — LIVE |
| 2 | `imersao-tools/docs/PRD-IMERSAO-ORIGINAI.md` | PRD unificado (baseado no todolist.html) |
| 3 | `docs/conceito-originai.md`, modulos, templates, prompts | Material de referencia ORIGINAI para Fase 3 |
| 4 (obsoleto) | `docs/roteiro-imersao-11-12-abril.md`, shards/ | OBSOLETOS — nao consultar |

## Se houver conflicto

Se um doc interno contradiz o todolist.html, o doc interno esta ERRADO. Corrigir o doc, nao perguntar ao Eurico.

## Origem desta regra

Incidente 07/04/2026 — PRD escrito com dados de docs internos errados (dizia "Sexta 18:00" quando era Sabado 10:00). 2 sessoes desperdicadas a debater um conflicto que nao existia.
